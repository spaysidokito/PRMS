# DEFINITIONS, ACRONYMS, AND ABBREVIATIONS

## Acronyms

**PRIMOSA**: Progressive Resource and Information Management System for Office of Student Affairs.

**OSA**: Office of Student Affairs.

## Definitions

**Agile Methodology**: A flexible and iterative development framework that emphasizes rapid feedback, continuous improvement, and user collaboration.

**Real-Time Collaboration**: A feature that allows multiple users to work on tasks or projects simultaneously, with updates reflected instantly for all participants.

**Automated Reporting**: A system feature that generates predefined reports automatically, providing insights into resource utilization, attendance, and event outcomes.

**Version Control**: A system that tracks and manages changes to files or tasks, ensuring smooth collaboration and preventing conflicts.

**Gantt Chart**: A visual planning tool used to display task timelines, dependencies, and deadlines.

**Cloud-Based System**: A system that operates on remote servers, allowing users to access data and tools via the internet.
